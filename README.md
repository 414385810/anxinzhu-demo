# 安心住 · 居家安全 AI 自查 Demo

用 AI 的眼睛帮您检查家里容易忽略的老人安全隐患。

## 功能特点

- **AI 视觉识别**：上传家中照片，AI 自动识别地面湿滑、通道障碍、照明不足、扶手缺失等安全隐患
- **风险分级**：按高/中/低风险分级展示，优先处理紧急问题
- **双版本报告**：
  - 子女版：专业评估报告，含安全评分、详细隐患清单、改造方案和沟通技巧
  - 父母版：暖心沟通版，语气温和不引起恐慌，适合直接发给爸妈看
- **改造预算**：根据预算档位推荐免费 DIY 和需要购买的改造项目
- **一键分享**：支持将两个版本报告分别保存为长图或复制文字，方便发给家人

## 演示路线

当前 Demo 覆盖"夜间起夜路线"：床 → 通道 → 卫生间

## 技术栈

纯前端单页应用，HTML + CSS + JavaScript，无后端依赖。
AI 识别能力接入 Qwen-VL-Plus（视觉识别）和 Doubao-Seed-2.1（交叉审查+报告生成），Demo 模式下使用预置数据。

## 本地运行

直接用浏览器打开 `index.html` 即可，或启动本地服务器：

```bash
python3 -m http.server 8080
```

然后访问 http://localhost:8080
